1 - git add xxx.html

Procédure pour descendre le projet :
1 - Se placer dans un dossier générique. Exemple : Documents ou Téléchargements
2 - git clone https://github.com/MarcoChampion4435/RFF.git
3 - File / Open folder / RFF
